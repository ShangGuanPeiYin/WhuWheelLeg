### 问题
逐飞的代码头文件结构是all .c包含 all .h，而.h包含Type头文件，不包含其他的头文件。这就会导致一个问题：当一个.h文件中需要其他.h文件的声明时，就会报错，原因是没有包含头文件。需要在.h中单独调用对应的.h文件。而.c文件中有会调用一遍，导致代码赘余。且包含了过多的无关的头文件，也会使得运行效率下降。
